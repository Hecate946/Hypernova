Files: Module for downloading files.

Dumpbans: DMs you a file of server bans.

Usage: -dumpbans
Aliases: -banlist, -txtbans
Permission: Ban Members
Output: Sends a txt file to your DMs with all past server bans.
Notes: If you have your DMs blocked, the bot will send the file to the channel where the the command was invoked.

Dumpbots: DMs you a file of server bots.

Usage: -dumpbots
Aliases: -botlist, -robotlist, -dumprobots
Permission: Manage Server
Output: Sends a txt file to your DMs with all bots currently in the server.
Notes: If you have your DMs blocked, the bot will send the file to the channel where the the command was invoked.
