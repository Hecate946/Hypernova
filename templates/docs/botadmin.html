Command_History: Show command history.

Usage: -command_history <option> [args]
Output: Recorded command history matching the specified arguments.
Options: command, server, user, log, cog

Debug: Show helpless commands.

Usage: -debug
Alias: -helpless
Output: Shows all commands that lack descriptions or help.

Del: Delete a message.

Usage -del
Output: Delete a specific message ID.

Dumpservers: DMs all servers.

Usage: -dumpservers
ALiases: -dumpguilds, {0]txtguilds, -txtservers
Permission: Bot Admin
Output: DMs you a file with all my servers, their member count, owners, and their IDs
Notes: If you have your DMs blocked, the bot will send the file to the channel where the the command was invoked.

Emojipost: Sends all server emojis.

Usage: -emojipost [nodm]
Alias: -epost
Output: Sends a formatted list of emojis and their IDs. Specify the nodm bool argument to avoid the bot from DMing you.

Extension: Show info on an extension.

Usage: -extension <extension>
Alias: -ext Outputs the cogs attatched to the passed extension.

Extensions: List all cogs.

Usage: -extensions
Alias: -exts
Output: Lists all extensions and their corresponding cogs.

Firstservers: Show first joined servers.

Usage: -firstservers
Output: Lists the first servers I joined

Gist: Post a gist on github.

Usage: -gist [filename=output.txt] [content]
Output: Creates a gist using your posted content
Notes: Will post a gist of the first message attachment with it's file extension if no content is passed.

Guild: Get stats on a bot server.

Usage: -guild [server]
Alias: -guildinfo, -gi
Output: Lists some info about the current or specified server.

Inv: Create a server invite.

Usage: -inv <server>
Output: Invite for that server (if bot could create it)

Listservers: Lists all servers

Usage: -listservers
Alias: -servers, -serverlist
Output: Lists the servers I'm connected to.

Members: Show members for a server.

Usage: -members
Output: Lists the members of a passed server

Message: DM a user the bot knows.

Usage: -message <user> <message>
Alias: -pm, -dm
Example: -dm 384059834985989 Hi
Permissions: Bot Owner

Nickscan: List all bot nicknames.

Usage: -nickscan
Alias: -ns
Output: All my nicknames across all servers

Recentservers: Show latest joined servers.

Usage: -recentservers
Alias: -lastservers
Output: Lists the most recent servers joined

Say: Say a message.

Usage: -say
Output: Deletes the command invocation and resends the exact content.

Sss: Show shared servers.

Usage: -sss [user]
Output: Servers that the user shares with the bot.
Notes: This supplies a more verbose output than the public ss command. Will default to you if no user is specified.

Topservers: Show most member servers.

Usage: -topservers
Output: The servers with the most memebers
