Voice: Module for voice permissions.

Djlock: Lock the music module to DJs.

Usage: -djlock
Alias: -musiclock
Output: Locks the music module to DJs

Djrole: Manage the DJ role.

Usage: -djrole [option]
Alias: -dj
Output: Manages the server DJ role
Options: - create # Creates the server DJ role - delete # Deletes the server DJ role
Notes: This role, when added to users, gives users full control over the music module. Note that all users with the Move Members, Mute Members, and Deafen Members permission will always be DJs.

Djrole Create: Create a DJ role.

Usage: -djrole create [role]
Alias: -djrole make
Output: Creates a DJ role
Notes: If no role is specified, a new role named DJ will be automatically created by the bot.

Djrole Delete: Delete the DJ role.

Usage: -djrole delete
Alias: -djrole remove
Output: Deletes the server DJ role

Djs: Show the server djs.

Usage: -djs
Output: Show all the server music djs and their respective statuses.

Djunlock: Unlock the music module.

Usage: -djunlock
Alias: -musicunlock
Output: Unlocks the music module to all

Vckick: Kick users from a voice channel.

Usage: -vckick <targets>...
Output: Kicks passed members from their channel
Permission: Move Members

Vcmove: Move a user from a voice channel.

Usage: -vcmove <targets>... <channel>
Output: Moves members into a new voice channel
Permission: Move Members

Vcpurge: Kick all users from a voice channel.

Usage: -vcpurge <voice channel>
Output: Kicks all members from the channel
Permission: Move Members

Vctransfer: Transfer users into a new channel.

Usage: -vctransfer <voice channel> <voice channel>
Output: Transfers all members from one channel to another
Permission: Move Members
