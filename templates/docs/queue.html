Queue: Module for managing the queue.

Clear: Clear the queue.

Usage: -clear
Alias: -c
Output: Removes all queued tracks.
Notes: Specify a range, e.g. -clear 3 10 to remove only the tracks between position 3 and 10 in the queue.

Dedupe: Remove duplicate tracks.

Usage: -dedupe
Alias: -dd, -uniquify, {deduplicate}
Output: Remove all duplicate tracks in the queue.

Dequeue: Clear a user's enqueues.

Usage: -dequeue [user]
Alias: -detrack, -dq
Output: Remove all tracks queued by a specific user.

Jump: Jump to a track in the queue.

Usage: -jump [index]
Output: Skips the current track and plays the selected track in the queue.
Notes: All other tracks in the queue will remain in their respective positions.

Loop: Loop the track or queue.

Usage: -loop [option]
Output: Loops the currently playing song.
Notes: Use -loop queue to loop the entire queue. Use -loop track to loop only the current track. Use -loop off to stop looping the track or queue. Will loop the current track if neither is specified. Use -unloop can also be used to stop looping settings.

Move: Move a song in the queue.

Usage: -move <index> <position>
Alias: -switch, -relocate
Output: Move a song to a new position in the queue.

Remove: Remove a track from the queue.

Usage: -remove [index]
Alias: -pop
Output: Removes a song from the queue at a given index.

Requeue: Queue the previous track.

Usage: -requeue
Alias: -last, -back, -previous
Output: Add the previous song to the end of the track queue.

Reverse: Reverse the queue.

Usage: -reverse
Output: Reverses the queue.

Shuffle: Shuffle the queue.

Usage: -shuffle
Output: Shuffles the queue.
Notes: Specify a range, e.g. -shuffle 3 10 to shuffle only the tracks between position 3 and 10 in the queue.

Skipto: Skip to a track in the queue.

Usage: -skipto [index]
Output: Skips the current track and plays the selected track in the queue.
Notes: Tracks before the selected song will be removed from the queue.

Tracks: Show the track queue.

Usage: -tracks
Alias: -q, -queue
Output: Starts a pagination session showing all the tracks in the current queue.
Notes: Each page contains 10 queue elements. Will invoke the play command if a search is specified.

Unloop: Un-loop the track or queue.

Usage: -unloop
Alias: -deloop
Output: Stops looping the current song or queue

Weed: Clear absent user enqueues.

Usage: -weed
Alias: -lc, -leavechannel, -cull
Output: Remove all tracks enqueued by users who have left the music channel.
