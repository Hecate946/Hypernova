Botconfig: Owner only configuration cog.

Addadmin: Add a new bot admin.

Usage: -addadmin <user>
Permission: Hecate#3523
Output: Adds the passed user's ID to the admins key in the config.json file
Notes: USE WITH CAUTION! This will allow the user to access global bot information. This includes s complete server list, member list, etc. To reflect changes instantly, use -botvars.

Addowner: Add a new bot owner.

Usage: -addowner <user>
Permission: Hecate#3523
Output: Adds the passed user's ID to the owners key in the config.json file
Notes: USE WITH CAUTION! This will allow the user to access all commands including those with root privileges. To reflect changes instantly, use the -botvars command

Adminlock: Admin-lock the bot.

Usage: -adminlock

Blacklist: Blacklist a discord object.

Usage: -blacklist <object> [reason]

Blacklisted: Show blacklisted objects.

Usage: -blacklisted
Output: Shows which users are blacklisted globally.

Change: Change the bot's specifications.

Usage: -change <options> <new>
Examples: -change name Tester -change avatar <url>
Permission: Bot Owner
Output: Edits the specified bot attribute.
Options: avatar, nickname, username status, activity, presence

Config: Change a config.json value.

Usage: -config <Key to change> <New Value>
Alias: -conf
Output: Edits the specified config.json file key to the passed value.
Notes: To reflect the changes of the file change immediately, use the -botvars command instead of rebooting the entire client.

Leaveserver: Have the bot leave a server.

Leaves a server - can take a name or id (owner only).

Ownerlock: Owner-lock the bot.

Usage: -ownerlock

Remadmin: Remove a bot admin.

Usage: -remadmin <user>
Aliases: -removeadmin, -rmadmin
Permission: Hecate#3523
Output: Removes a user from the admins key in my config.json file
Notes: To reflect changes instantly, use the -botvars command

Remowner: Remove a user from my owner list.

Usage: -remowner <user>
Aliases: -removeowner, -rmowner
Permission: Hecate#3523
Output: Removes a user from the owners key in my config.json file
Notes: To reflect changes instantly, use the -botvars command

Toggle: Toggle disabling a command.

Usage: -toggle <command>
Output: Globally disables or enables a command

Unblacklist: Unblacklist discord objects.

Usage: -unblacklist <object>

Write: Write to the bot overview or changelog.

Usage: -write <method>
Aliases: -set, -add
Methods:
overview: Edit the bot's overview
changelog: Post an entry to the changelog

Write Readme: Update the bot readme file.

Usage: -readme
Alias: -md
Output: Sends my readme file on github to your DMs
Notes: This command updates the readme file to include all the current command descriptions for each registered category.
