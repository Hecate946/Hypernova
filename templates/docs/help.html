Help: My extensive help category.

Aliases: Show the aliases for a command

Usage: -aliases <command>
Alias: -alias
Output: Shows all the aliases for a given command

Botperms: Check if the bot can run a command.

Usage: -botperms <command>
Aliases: -bothasperms, -botpermissions, -bothaspermissions
Output: Check which permissions the bot requires to run that command.
Notes: The emote will signify whether or not the bot has the permissions needed to run that command.

Brief: Get the short description of a command.

Usage: -brief <command>
Aliases: -shortdocs, -shortdoc
Output: The short description of the passed command

Canrun: See if a command can be executed.

Usage: -canrun <command>
Alias: -checker
Output: Tells you whether or not you have permission to run a command

Category: Show the parent category of a command.

Usage: -category <command>
Alias: -cog <command>
Output: Show the parent category (cog) of a passed command.

Commandinfo: Get attribute info on a command.

Usage: -commandinfo <command>
Alias: -cmdinfo
Output: Specific command information

Commandsearch: Search for a command by name.

Usage: -commandsearch <search>
Alias: -cmdsearch
Output: Searches for the most similar commands based off a search query. Outputs the results in tabular format.

Cooldown: Show the cooldown for a command.

Usage: -cooldown <command>
Output: Show the cooldown for a command.

Docstring: Get the help docstring of a command.

Usage: -docstring <command>
Alias: -helpstr <command>
Output: The long description of the passed command

Examples: Get specific examples for a command.

Usage: -example <command>
Alias: -ex
Output: Shows all possible command usages with aliases and valid arguments.

Help: My documentation for all commands.

Made: Show when a command was first made.

Usage: -made <command>

Updated: Show when a command was last updated.

Usage: -updated <command>
Alias: -changed
Output: Show exactly when a command was last updated by the owner.

Usage: Get a usage example for a command.

Usage: -usage <command>

Usage: -where <command>
