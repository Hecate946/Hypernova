Playlists: Module for managing playlists.

Playliked: Enqueue saved songs.

Usage: -playliked
Alias: -playsaved
Output: Enqueues all tracks in the your liked songs by adding them to the end of the current queue.
Notes: Add to your saved songs by using the -save command

Playlist Append: Add to a saved playlist.

Usage: -playlist append [playlist name]
Output: Appends the current song to the end of a saved playlist.

Playlist Delete: Delete one of your saved playlists.

Usage: -playlist delete [playlist name]
Output: Deletes a saved playlist by name.

Playlist Enqueue: Enqueue a saved playlist.

Usage: -playlist enqueue [playlist name]
Aliases: -playlist play, -playlist start, 0}playlist queue, 0}playlist load
Output: Enqueues all tracks in the saved queue by adding them to the end of the current queue.
Notes: Save playable queues by using the -playlist save command.

Playlist Extend: Extend a saved playlist with the current queue.

Usage: -playlist save [playlist name]
Output: Saves the current queue to a playlist with a given name.
Notes: This command will attach the current queue and the provided name to your ID. You will be the only person able to playback this queue at a later time.

Playlist Save: Save the queue to a new playlist.

Usage: -playlist save [playlist name]
Output: Saves the current queue to a playlist with a given name.
Notes: This command will attach the current queue and the provided name to your ID. You will be the only person able to playback this queue at a later time.

Playlist View: Show info on a playlist.

Usage: -playlist view [playlist owner] [playlist name]
Alias: -playlist info
Output: Show some statistics on a saved playlist

Playlists: Show queues saved by a user.

Usage: -queues [user]
Aliases: -myqueues, -savedqueues, -userqueues
Output: Shows all saved queues of a user in a pagination session.
Notes: Save playable queues by using the -savequeue command.

Playtop: Enqueue the top 10 most frequently queued tracks.

Usage: -playliked
Alias: -playsaved
Output: Enqueues all tracks in the your liked songs by adding them to the end of the current queue.
Notes: Add to your saved songs by using the -save command
