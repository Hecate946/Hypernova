Stats: Module for server statistics

Admins: Show the server admins.

Usage: -admins
Alias: -administrators
Output: Show all server administrators and their respective statuses.

Channelinfo: Get info about a channel.

Usage: -channelinfo [channel]
Output: Specific info on a given channel
Notes: If no channel is specified, current channel will be used.

Emojiinfo: Get info about an emoji.

Usage: -emojiinfo [emoji]
Aliases: -ei, -emoteinfo
Output: Specific info on a given channel
Notes: If no channel is specified, current channel will be used.

Emojistats: Emoji usage tracking.

Usage: -emojistats [user]
Alias: -estats
Permission: View Audit Log
Output: Get detailed emoji usage stats.
Notes: Specify an optional user to narrow statistics to be exclusive to the user.

Emojiusage: Get usage stats on an emoji.

Usage: -emojiusage <custom emoji>
Aliases: 0}emojiusage, -emoteusage
Output: Usage stats on the passed emoji

Emptyroles: Show roles that have no users.

Usage: -emptyroles
Output: Shows all roles with zero users

Firstjoins: Show the first users to join.

Usage: -firstjoins
Alias: -earlyjoins
Permission: View Audit Log
Output: Embed of all users ordered by their join date earliest to latest.

Joined: Check when a user joined the server.

Usage: -joined [user]
Aliases: -joinedat, -jointime
Permission: View Audit Log
Output: Shows when the passed user joined the server
Notes: Will default to youself if no member is passed.

Joinedatpos: Show who joined at a position.

Usage: -joinedatpos <position>
Alias: -joinedatposition
Permission: View Audit Log
Output: Shows the user that joined at the passed position.

Joinpos: Show the join position of a user.

Usage: -joinpos [user]
Alias: -joinposition
Permission: View Audit Log
Output: Shows when a user joined compared to other users.
Notes: Will default to yourself if no user is passed.

Lastjoins: Show the latest users to join.

Usage: -lastjoins
Aliases: -latestjoins, -recentjoins
Permission: View Audit Log
Output: Embed of all users ordered by their join date latest to earliest.

Listbots: Shows all the server's bots.

Usage: -listbots
Aliases: -allbots -serverbots -guildbots
Permission: View Audit Log
Output: A pagination session showing all the server's bots and their IDs.

Listchannels: Show the server's channels.

Usage: -listchannels
Alias: -channels, -allchannels, -serverchannels
Permission: Manage Channels
Output: Embed of all the server's channels

Listroles: Show an embed of all server roles.

Usage: -listroles
Alias: -roles, -allroles
Permission: Manage Roles
Output: Shows roles and their member counts. Takes one argument, sort_order, which can be default, name, count, or color.

Mods: Show the server moderators.

Usage: -mods
Alias: -moderators
Output: Show all the server moderators and their respective statuses.

Permissions: Show a user's permissions.

Usage: -permissions [user] [channel]
Alias: -perms
Output: Shows a user's permissions in a specific channel.
Notes: Will default to yourself and the current channel if they are not specified.

Rolecall: Counts the users with a role.

Usage: -rolecall <role>
Output: Shows the number of people with the passed role.

Rolecount: Counts the roles on the server.

Usage: -rolecount
Output: Counts all server roles

Roleinfo: Get information on a role.

Usage: -roleinfo <role>
Alias: -ri
Output: Shows details on the role's color, creation date, users, and creator.

Roleperms: Show the permissions for a role.

Usage: -roleperms <role>
Alias: -rp, -rolepermissions
Output: Embed with all the permissions granted to the passed role

Serverinfo: Show server information.

Usage: -serverinfo
Aliases: -serverinfo -serverstats -server -si -ss
Output: General stats on the server. Shows the name, ID, owner, user count, online member, role count, region, emoji count, channel info, and creation date.

Topic: Show the topic of a channel.

Usage: -topic [channel]
Alias: -channeltopic
Output: Shows a channel's topic. Useful for replying to users not abiding by the channel's topic.
Notes: Will default to the current channel if not alternative channel is passed.

Userinfo: Show information on a user.

Usage: -userinfo <member>
Aliases: -profile, -ui, -whois
Output: General stats on a discord user. Includes, ID, name, highest role, status, messages sent, commands run join position, and registration date.
Notes: Moderators, incoke command with -user, -rawuser, or -lookup to see all information currently available on a discord user.

Usage: -whohas <role>
Output: Lists the people who have the specified role with their status.
