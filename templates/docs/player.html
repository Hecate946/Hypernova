Player: Module for playing audio.

Connect: Connect to a channel.

Usage: -connect [channel]
Alias: -join
Output: Connects to a specified voice or stage channel.
Notes: If you do not specify a channel, the bot will join your current channel. (If it exists) If the bot is currently active in another channel, you must have DJ perms for the command to function properly.

Current: Show track info.

Usage: -current
Aliases: -now, -np, -current, -nowplaying
Output: Displays the currently playing song.
Notes: Will fall back to sending a codeblock if the bot does not have the Embed Links permission. A progress bar will be included if the bot has the Attach Files permission.

Disconnect: Disconnects from a channel.

Usage: -leave
Alias: -dc, -disconnect
Output: Clears the queue and leaves the voice or stage channel.

Fastforward: Fast forward the track.

Usage: -fastforward [seconds]
Alias: -ff
Output: Fast forward a certain number of seconds in a song.

Pause: Pause the track.

Usage: -pause
Alias: -halt
Output: Pauses the current track.
Notes: To resume playback, use the -resume command or invoke the -play command with no arguments.

Play: Play a track from a search.

Usage: -play <search>
Alias: -p, -enqueue
Output: Plays a track from your search.
Notes: If there are tracks in the queue, this will be queued until the other tracks finished playing. This command automatically searches youtube if no url is provided. Accepts spotify and youtube urls.

Playnext: Front queue a track.

Usage: -playnext <search>
Alias: -frontqueue, -pn
Output: Plays a track from your selection.
Notes: If there are tracks in the queue, this will be queued before all other tracks in the queue. If the track you select is a playlist, all tracks in the playlist will be placed ahead of the previously queued tracks.

Position: Show the track position.

Usage: -position
Alias: -pos
Output: Shows the current position of the song.

Replay: Play the previous track.

Usage: -replay
Alias: -again
Output: Add the previous song to the beginning of the track queue.

Resume: Resume the track.

Usage: -resume
Alias: -unpause
Output: Resumes playback paused by the -pause command.

Rewind: Rewind the track.

Usage: -rewind [seconds]
Alias: -rw
Output: Rewind a certain number of seconds in a song.

Save: Save a song to your liked songs.

Usage: -save
Aliases: -like
Output: Sends info and saves the current track
Notes: Will attempt to DM you the info and fall back to sending track information to the current channel. Saving songs with this command will add songs to your "Liked Songs" which can be played using the playliked command.

Seek: Seek to a position in the track.

Usage: -seek [time]
Output: Seeks to a certain position in the track
Notes: The position must be given in seconds.

Skip: Skip the track.

Usage: -skip
Aliases: -s, -fs, -vs, -next -voteskip, -forceskip
Output: Vote to skip a song.
Notes: The song requester and those with the Manage Server permission can automatically skip Otherwise half the listeners neet to vote skip for the song to be skipped.

Stop: Stop track and clear the queue.

-Usage: -stop
Output: Stops playing song and clears the queue.

Trackinfo: Show track info.

Usage: -trackinfo
Aliases: -now, -np, -current, -nowplaying
Output: Displays the currently playing song.
Notes: Will fall back to sending a codeblock if the bot does not have the Embed Links permission. A progress bar will be included if the bot has the Attach Files permission.

Youtube: Get results from a search.

Usage: -youtube <search>
Alias: -yt, -ytsearch
Output: Searches youtube and returns an embed of the first 10 results collected.
Notes: Choose one of the titles by typing a number or cancel by typing "cancel". Each title in the list can be clicked as a link.
