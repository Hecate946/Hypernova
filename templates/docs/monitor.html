Monitor: Module for monitoring bot status.

Bothealth: Show bot health.

Usage: -bothealth
Output: Various bot health monitoring tools.

Logger: View logging files.

Usage: -logger <option>
Alias: -l
Permission: Bot owner
Output: View any log recorded in ./data/logs
Options: commands|command|cmds|cmd errors|stderr|error|err|e information|info|i tracebacks|traceback|trace|tb|t clear|clr|cl

Logger Clear: Delete a logging file.

Usage: -logger clear <search>
Aliases: -cl, -clr
Output: Deletes a logging file.
Searches: commands|command|cmds|cmd errors|stderr|error|err information|info|i tracebacks|traceback|trace|t

Logger Commands: Show the commands.log file.

Usage: -logger commands
Aliases: -logger cmds
Output: Starts a pagination session showing the commands.log file

Logger Errors: Show the errors.log file.

Usage: -logger errors
Aliases: -logger e -logger err -logger error -logger stderr
Output: Starts a pagination session showing the errors.log file

Logger Info: Show the info.log file.

Usage: -logger info
Aliases: -logger i -logger information
Output: Starts a pagination session showing the info.log file

Logger Music: Show the music.log file.

Usage: -logger errors
Aliases: -logger m -logger music -logger audio -logger voice
Output: Starts a pagination session showing the errors.log file

Logger Traceback: Show the traceback.log file

Usage: -logger traceback
Aliases: -logger t -logger tb -logger trace -logger tracebacks
Output: Starts a pagination session showing the traceback.log file

Objgrowth: Debug memory leaks.

Usage: -objgrowth
Alias: -objg
Output: Shows detailed object memory usage

Pm2: View pm2 files.

Usage: -pm2 <option>
Output: View any pm2 log file in ./data/pm2
Options: stdout|out|output stderr|err|error|errors pid|process|processid clear|clr|cl

Pm2 Clear: Delete a pm2 log file.

Usage: -pm2 clear <search>
Aliases: -cl, -clr
Output: Deletes a pm2 file.
Searches: out|output|stdout errors|stderr|error|err pid|process|processid

Pm2 Pid: View the pm2 process ID.

Usage: -pm2 pid
Aliases: -pm2 process -pm2 processid
Output: Shows the process ID of the pm2 process.

Pm2 Stderr: View the pm2 stderr file

Usage: -pm2 stderr
Aliases: -pm2 err -pm2 error -pm2 errors
Output: Starts a pagination session showing the pm2 stderr file.

Pm2 Stdout: View the pm2 stdout file.

Usage: -pm2 stdout
Aliases: -pm2 out -pm2 output
Output: Starts a pagination session showing the pm2 stdout file.

Threadinfo: Show bot threadinfo.

Usage: -threadinfo
Output: Shows some info on bot threads.

Todo: Manage the bot's todo list.

Usage: -todo <method>
Alias: -to-do
Methods: no
subcommand: shows the todo list
add: Adds an entry to the todo list
remove|rm|rem: Removes an entry from the todo list
