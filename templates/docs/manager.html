Manager: Manage bot processes and cogs.

Botvars: Reload the bot variables.

Usage: -botvars
Output: Resets all global bot variables
Notes: Useful to use this command as an alternative to restarting the bot when changes are made to the config.json file

Cleartrace: Clear the console.

Usage: -cleartrace
Alias: -cl
Output: Clears the console and prints a clean nicely formatted message.

Do: Repeat a command.

Usage: -do <times> <command>
Output: Repeats a command a specified number of times.

Elapse: Time a command response.

Checks the timing of a command, attempting to suppress HTTP and DB calls.

Eval: Evaluate python code.

Usage: -eval <body>
Aliases: -e -exe -exec -evaluate
Output: Code evaluation
Environment:
"self": Manager,
"utils": utils,
"converters": converters,
"discord": discord,
"bot": self.bot,
"ctx": ctx,
"channel": ctx.channel,
"author": ctx.author,
"guild": ctx.guild,
"message": ctx.message,
"_": self._last_result,

Git: Run github commands.

Usage: -git <command>
Alias: -github
Output: Runs a git command.
Notes: Use the shorthand -git give to run 3 commands concurrently. namely, git add . git commit -m "update" git push

Load: Load an extension.

Usage: -load <file>
Aliases: -loadcog, -lc
Output: Loads the extension (cog) with the given name.

Ras: Reload all settings modules.

Usage: -ras
Aliases: -reloadallsettings
Output: Reloads all extensions in the ./settings directory. Excludes database.py.

Rau: Reload all utilities modules.

Usage: -reloadallutils
Aliases: -rau
Output: Reloads all extensions in the ./utilties directory.

Reboot: Cleanly reboot the bot.

Usage: -reboot
Alias: -restart
Output: Shuts down all processes, sends a confirmation message, then updates the message when the bot is once again ready.

Refresh: A faster way of rebooting.

Usage: -refresh [db|-db|--db=False]
Output: Runs all reloading commands.
Notes: Pass the --db flag to restart the database.

Reload: Reload an extension.

Usage: -reload <file>
Alias: -r
Output: Reloads the extension (cog) with the given name.

Reloadall: Reload all extensions.

Usage: -reloadall
Alias: -ra
Output: Reloads all extensions in the ./cogs directory.

Reloadsetting: Reload a utilities module.

Usage: -reloadutil <file>
Aliases: -rss -reloadsettings
Output: Reloads an extension in the ./settings directory.

Reloadutil: Reload a utilities module.

Usage: -reloadutil <file>
Aliases: -ru -reloadutils -reloadutility
Output: Reloads an extension in the ./utilities directory.

Sh: Run shell commands.

Runs a shell command.

Sudo: Run a command as another user.

Usage: -sudo [channel] [user] <command>
Outpue: Run a command as another user optionally in another channel.

Unload: Unload an extension.

Usage: -unload <file>
Aliases: -unloadcog, -uc
Output: Unloads the extension (cog) with the given name.
