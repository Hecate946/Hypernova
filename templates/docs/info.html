Info: Module for bot information.

About: Display information about the bot.

Usage: -about
Aliases: -info, -bot, -botstats, -botinfo, -ab
Output: Version info and bot stats

Avgping: View the average message latency.

Usage: -avgping
Aliases: -averageping -avglat -avglatency
Output: Shows the average message latency over the past 500 messages sent.

Botadmins: Show the bot's admins.

Usage: -botadmins
Alias: -badmins
Output: An embed of all the current bot admins

Botowners: Show the bot's owners.

Usage: -botowners
Alias: -owners
Output: An embed of the bot's owners

Bugreport: Send a bugreport to the developer.

Usage: -bugreport <report>
Aliases: -issuereport, -reportbug, -reportissue
Examples: -bugreport Hello! I found a bug with this command...
Output: Confirmation that your bug report has been sent.
Notes: Do not hesitate to use this command, but please be very specific when describing the bug so that the developer may easily see the issue and correct it as soon as possible.

Changelog: Show my changelog.

Usage: -changelog
Output: My changelog

Cogs: List all current cogs.

Usage: -cogs
Output: An embed of all my current cogs

Github: Github commit history.

Usage: -github
Alias: -updates
Output: Shows an embed of the last 5 github commits

Hostinfo: Show the host environment.

Usage: -hostinfo
Output: Detailed information on the bot's host environment

Invite: Invite the bot to your server!

Usage: -invite
Aliases: -bi, -botinvite
Output: A selection of invite links to invite the bot to your server

Lines: Show sourcecode statistics.

Usage: -lines
Aliases: -cloc, -code, -codeinfo
Output: Gives the linecount, characters, imports, functions, classes, comments, and files within the source code.

Neofetch: Run the neofetch command.

Usage: -neofetch
Alias: -nf
Output: Some stats on the bot's server

Overview: Show some info on the bot's purpose.

Usage: -overview
Alias: -purpose
Output: Me and my purpose

Ping: Test the bot's response latency.

Usage: -ping
Aliases: -latency, -response
Output: Bot latency statistics.
Notes: Use -speed and the bot will attempt to run an internet speedtest. May fail.

Privacy: View the privacy policy.

Usage: -privacy
Alias: -policy
Output: Details on terms of use and privacy regarding the bot and its database.

Replytime: Show reply latencies.

Usage: -replytime
Output: Shows 3 times showing the discrepancy between timestamps.

Sharedservers: Show servers shared with the bot.

Usage: -sharedservers [user]
Alias: -shared
Output: Counts the servers in common between the user passed and the bot.
Notes: Will default to youself if no user is passed

Socket: Show bot socket stats.

Usage: -socket
Alias: -socketstats
Output: Fetch information on the socket events received from Discord.

Source: Display the source code.

Usage: -source [command]
Alias: -sourcecode, -src
Notes: If no command is specified, shows full repository

Stats: Display the bot's discord info.

Usage: -about
Aliases: -info, -bot, -botstats, -botinfo, -statistics
Output: Version info and bot stats

Suggest: Send a suggestion to the developer.

Usage: -suggest <report>
Alias: -suggestion
Examples: -suggest Hello! You should add this feature...
Output: Confirmation that your suggestion has been sent.
Notes: Do not hesitate to use this command, your feedback is valued immensly. However, please be detailed and concise.

Support: Join my support server!

Usage: -support
Aliases: -sup, -assist, -assistance
Output: An invite link to my support server

Uptime: Show the bot's uptime.

Usage: -uptime
Alias: -runtime
Output: Time since last boot.

Uptimeinfo: Show a graph of uptime stats

Usage: -uptimeinfo
Aliases: -percentuptime, -pieuptime, -pu
Output: Shows a pie-chart graph of bot uptime across all time

Users: Show users I'm connected to.

Usage: -users
Aliases: -userstats, -usercount
Output: Shows users and bots I'm connected to and percentages of unique and online members.

Vote: Show your support by voting for me!

Usage: -vote
Output: A link to top.gg where you can vote to support me.
