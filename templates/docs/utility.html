Utility: Module for general utilities.

Ascify: Convert special characters to ascii.

Usage: -ascify <string/member>
Aliases: -ascii, -normalize
Output: Attempts to convert a string or member's nickname to ascii by replacing special characters.
Notes: If the passed argument is a user and both the command executor and the bot have the required permissions, the bot will set the user's nickname to the ascified version of the word. Otherwise, it will simply return the ascified version. If the passed string is already in ASCII, it will return the same result.

Avatar: Show a user's avatar.

Usage: -avatar [user]
Aliases: -av, -pfp, -icon
Examples: -avatar 806953546372087818, -avatar Neutra
Output: Shows an enlarged embed of a user's avatar.
Notes: Will default to you if no user is passed.

Badges: Show all the badges a user has

Usage: -badges [user]
Alias: -flags
Output: Shows all the badges a user has
Notes: Will default to you if no user is passed.

Banner: Show a user's banner.

Usage: -avatar [user]
Aliases: -av, -pfp, -icon
Examples: -avatar 810377376269205546, -avatar Neutra
Output: Shows an enlarged embed of a user's avatar.
Notes: Will default to you if no user is passed.

Calculate: Calculate a math formula.

Usage: calculate <formula>
Aliases: -math, -calc
Output: The caluculated result of your input formula
Keys:
exponentiation: '^'
multiplication: 'x' | '*'
division: '/'
addition: '+' | '-'
integer: ['+' | '-'] '0'..'9'+
constants: PI | E
Functions: # To be used in the form -calc function(expression) sqrt, log, sin, cos, tan, arcsin, arccos, arctan, sinh, cosh, tanh, arcsinh, arccosh, arctanh, abs, trunc, round, sgn

Charinfo: Show information on a character.

Usage: -charinfo <characters>
Alias: -unicode
Output: Shows information on the passed characters.
Notes: Maximum of 25 characters per command.

Clock: Get the time of any location

Usage: -clock <place>
Aliases: -worldclock, -worldtime
Output: Shows the current time of day it is in the specified location.
Notes: Can accept cities, states, provinces, and countries as valid locations.

Color: Show a given color and its values.

Usage: -color <value>
Output: View info on a rgb, hex or cmyk color and their values in other formats
Notes: Will try to convert value into role and return role color before searching for hex, decimal, rgb, and cmyk

Defaultavatar: Show a user's default avatar.

Usage: -defaultavatar [user]
Aliases: -dav, -dpfp, -davatar
Output: Shows an enlarged embed of a user's default avatar.
Notes: Will default to you if no user is passed.

Dehoist: Dehoist a specified user.

Usage: -dehoist <user>
Permission: Manage Nicknames
Output: Re nicknames a single user who hoists themselves at the top of the member list by using special characters
Notes: To dehoist all users with a single command, use -massdehoist. If the bot or the command author lack permissions to edit a nickname, the bot will output a dehoisted version of the target user's name.

Embed: Create an embed interactively.

Usage: -embed
Alias: -embedder
Permissions: Manage Messages, Embed Links
Output: Starts an interactive embed creating session.
Instructions: Use "cancel" at any time to cancel the session Use "none" at any time to skip the value Use "end" at any time to end the session and send the embed.
Notes: If the bot has the Manage Messages permission, it will prompt you for a cleanup option after completion. This will purge all messages sent as a result of this embed session and leave the embed.

Geninv: Generate a server invite link.

Usage: -geninvite [bot] [permissions]
Aliases: -geninv -generateinv -generateinvite
Output: Generates a server invite link with your specified channel, uses and expiration date.
Notes: Will create a permanent invite link with unlimited uses if no channel, uses, or expiration date is specified.

Gtoken: Generate a discord token.

Usage: -gtoken <user>
Aliases: -gt, -generatetoken
Output: Generates a discord token for a user
Notes: Defaults to you if no user is passed.

Nickname: Edit or reset a user's nickname

Usage: -nickname <user> [nickname]
Aliases: -nick, -setnick
Permission: Manage Nicknames
Output: Edits a member's nickname on the server.
Notes: Nickname will be reset if no new nickname is passed.

Oauth: Generate a bot invite link.

Usage: -oauth [bot] [permissions]
Aliases: -oauth2 -genoauth -genbotoauth
Output: Generates a bot invite oauth URL with your specified permissions.
Notes: Defaults to me if no bot is specified.

Platform: Show a user's discord platform.

Usage: -platform [user]
Alias: -mobile, -desktop, -web, -device
Output: Shows which discord platform a user is currently on. Can be discord desktop, discord mobile, or discord web.
Notes: The bot cannot determine platform when users are offline or if their status is invisible.

Ptoken: Decode a discord token.

Usage: -ptoken <token>
Aliases: -pt, -parsetoken
Output: Decodes the token by splitting the token into 3 parts.
Notes: First part is a user id where it was decoded from base 64 into str. The second part is the creation of the token, which is converted from base 64 into int. The last part cannot be decoded due to discord encryption.

Raw: Shows the raw content of a message.

Usage: raw [message id]
Alias: -content
Output: Raw message content

Usage: -snowflake <id>
Aliases: -id, -age
Output: The exact date & time that the discord snowflake was created
Notes: Will calculate when the snowflake will be created if it does not yet exist.

Status: Show a user's discord status.

Usage: -status [user]
Aliases: -game, -playing, -activity, -presence

Notes: Will default to you if no user is passed.
